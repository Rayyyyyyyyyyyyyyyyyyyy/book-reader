# Rui-Xuan 生命素材（讀書心得的血肉來源）

> 這頁只給 `/rui-xuan-book` 讀，不發佈。
> 用法：寫到某個抽象概念、需要一個具體例子時，從這裡挑一條真實的事，再用「你／我們」改寫成普世句。
> 私下記錄是真實的；輸出時抽象化，不指名道姓。

| 主題（真實的事） | 可接到的概念 | 落地句範例（你／我們，句尾不留標點） |
|---|---|---|
| 那段早就該結束、卻放不下的關係 | 沉沒成本、損失趨避、執著、面子 | 那段早就該結束的關係，我們還是繼續往裡面丟自己 |
| 一個人去日本，去了五次以上 | 自我重整、獨處、找回主體性 | 有些路要一個人走，才聽得見自己原本的聲音 |
| 同事不思考、把責任推給你 | ownership、把腦袋外包、成熟 | 把腦袋外包出去的人，總有一天要別人替他收拾 |
| 養貓 ㄤ堆（膽小、社恐、無條件） | 穩定的情感、被需要、不求回報的陪伴 | 牠不懂你今天值不值得被愛，牠只是靠過來 |
| 僵直性脊椎炎、心悸、睡不好、疲勞 | 身體的有限、與身體和解、活在當下 | 身體會在你最不想聽的時候，提醒你它也有意見 |
| 演唱會票券司法、帳戶凍結、等不起訴 | 失控、制度、資源被卡住 | 有些失控不是你的錯，但收拾的還是你 |
| 想要空間、又極度想被穩定地愛 | 自由 vs 歸屬、安全地連結 | 我們說自己要的是自由，其實要的是能安全地靠近 |
| 把每次痛苦整理成文字、表格、分析 | 理智化、用秩序對抗混亂 | 把痛苦拆成條列，是因為整齊的東西比較不會痛 |
| 大量用 AI 重構工作、做內部訓練 | 工具、效率、思考密度 | 工具換了，問題沒換：你還是得自己決定要往哪想 |
