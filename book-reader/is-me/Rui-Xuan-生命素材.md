# Rui-Xuan 生命素材（讀書心得的血肉來源）

> 這頁只給 `/rui-xuan-book` 讀，不發佈。
> 用法：寫到某個抽象概念、需要一個具體例子時，從這裡挑一條真實的事，再用「你／我們」改寫成普世句。
> 私下記錄是真實的；輸出時抽象化，不指名道姓。

| 主題（真實的事） | 可接到的概念 | 落地句範例（你／我們，句尾不留標點） |
|---|---|---|
| 那段早就該結束、卻放不下的關係 | 沉沒成本、損失趨避、執著、面子 | 那段早就該結束的關係，我們還是繼續往裡面丟自己 |
| 一個人去日本，去了五次以上 | 自我重整、獨處、找回主體性 | 有些路要一個人走，才聽得見自己原本的聲音 |
| 同事不思考、把責任推給你 | ownership、把腦袋外包、成熟 | 把腦袋外包出去的人，總有一天要別人替他收拾 |
| 養貓 ㄤ堆（膽小、社恐、無條件） | 穩定的情感、被需要、不求回報的陪伴 | 牠不懂你今天值不值得被愛，牠只是靠過來 |
| 僵直性脊椎炎、心悸、睡不好、疲勞 | 身體的有限、與身體和解、活在當下 | 身體會在你最不想聽的時候，提醒你它也有意見 |
| 演唱會票券司法、帳戶凍結、等不起訴 | 失控、制度、資源被卡住 | 有些失控不是你的錯，但收拾的還是你 |
| 想要空間、又極度想被穩定地愛 | 自由 vs 歸屬、安全地連結 | 我們說自己要的是自由，其實要的是能安全地靠近 |
| 把每次痛苦整理成文字、表格、分析 | 理智化、用秩序對抗混亂 | 把痛苦拆成條列，是因為整齊的東西比較不會痛 |
| 大量用 AI 重構工作、做內部訓練 | 工具、效率、思考密度 | 工具換了，問題沒換：你還是得自己決定要往哪想 |
| 面試被問「三年後的自己在哪」，答不出來，只說可能會有個女朋友，沒拿到 offer | 規劃的幻覺、資訊不足下被要求建構願景、誠實 vs 標準答案 | 要我們用幾頁官網和幾個記不住的名字，拼出三年後的自己，本來就強人所難 |
| 現在重想三年後還是不知道，只想跟三年前的自己說傾家蕩產也要 all in 台積電 | 事後諸葛 vs 當下無知、科技變化太快、規劃趕不上現實 | 我們以為的看清楚，多半只是站在現在，回頭嘲笑當時看不見的自己 |
| 一個人去好市多，觀察陌生人的互動、偷看別人推車，孤獨但平靜富足 | 孤獨作為存在狀態 vs 情緒、活在當下、自我完整 | 孤獨不是一種情緒，是一種存在狀態，情緒取決於你在裡面怎麼活動 |
| 在長期不對等的關係裡壓低自己、後來學會設停損點 | 選擇權、主導權、不對等、停損不是冷漠 | 因為能走，所以留下才是自願，而不是消耗 |
| 渴望被需要、被依賴、被重視，卻難以自我認同 | 被看見 vs 自我認同、童年情感忽視、向外求認可 | 我們急著要別人的重視，其實是還沒練會自己看見自己就夠了 |
| 用海浪與船槳想像面對問題的心法 | 正念、專注當下、覺察情緒、谷底總會過去 | 浪一波波來，你能做的是專注在手上那支槳，一次只划一下 |
| 學著把自己當成一個正在認識的戀愛對象來對待 | 愛自己、自我接納、不急著改造自己 | 每多理解自己一點，就多靠近自己一點 |
| 自認是普通人，但回首過去覺得自己很幸運，貴人總在需要時剛好出現 | 幸運、回頭才看懂、身在其中看不見 | 有些貴人是回頭才認得出來的，當下你只是剛好被接住 |
| 「喜歡現在的自己」的時刻都是小事：晚餐調味剛好、聽音樂搖擺、貓吃得開心、照鏡子覺得好看 | 自我喜歡、日常的富足、活在當下 | 喜歡自己常常不是什麼大成就，是晚餐剛好、是鏡子裡的自己還不錯 |
| 把面對過去當說書人／導演，記憶每次回想都會被重新寫入 | 敘事自我、記憶可重新整理、用新理解看過去 | 過去改不了，但你能決定用什麼情緒把它說成一個故事 |
| 搭 2am 的飛機，在雲海上被太陽曬醒看日出 | 體驗不折舊、魔幻時刻、活著的當下 | 有些風景是熬夜換來的，太陽從雲海冒頭那刻，像被整個世界叫醒 |
| 對外貌紅利與擇偶公式的憤怒（七龍珠條件、投射父母形象） | 不公平、外貌紅利、投射、條件與內在無關 | 那張清單上的每一條，都跟一個人內在的好壞完全無關 |
| 從「找填補空缺的人」變成「找能一起生活的人」；會有期待但那是自己的課題 | 課題分離、不要求對方改變、配速、成熟的愛 | 我有我的富足，你有你的人生，我只是想看看一起走一段會是什麼風景 |
