## Book Notes Format

Each book note follows a loose structure:
- H1 title = book name
- H2 sections = thematic takeaways (not chapter summaries)
- Written in first-person reflective Chinese prose
- Tone: introspective, emotionally grounded, connected to lived experience

When drafting or editing book notes, match this style — not academic, not listicle, but personal synthesis.

## is-me/ Structure

- `Rui-Xuan.md` — ongoing character analysis, updated after each batch of chat reading. Contains 12-dimension profile and per-round session notes.
- `chat/` — LINE chat exports (`.txt` files) used as source material for the analysis

The analysis task: read chat files in 2000-line batches, update `Rui-Xuan.md` after each batch, accumulating a layered understanding of the person's speech patterns, behavioral tendencies, and inner evolution.
